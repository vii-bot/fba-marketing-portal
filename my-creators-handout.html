Admin Controls

Add Creator

3-tab modal: Info (code, name, status, priority, runners, folders, overview) · Niche (niche, aesthetic, styles) · Assets (physical attributes, strengths, weaknesses).

Code + Name required to save.

Edit Creator

Available from the Creator Profile header. Pre-fills all three tabs with existing data for updating any field.

Archive Creator

Hides the creator from all active views. Data is preserved — archiving is not deletion.

Key Reminders for Page Runners

📋

Check the Overview tab before your shift — media folder links, niche, and brief are there.

✍️

Read the Strategy tab before writing captions — every creator has their own tone guide.

⭐

Log Highlights when a post performs well — don't skip this step.

⚠️

A Needs Media badge means report to management on Discord right away.

📝

Use Notes to flag anything unusual — account issues, creator concerns, or content blockers.

🔍

Use the search bar to find creators fast — search your own username to see all of yours.
